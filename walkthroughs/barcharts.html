Total riders by city via bar chart

Goal: create a bar chart showing total riders per city across all years and modes. This is a classic dashboard component and works well with a city filter later.

Key idea

For a bar chart, we usually want one row per category. That means we need to group by city and sum riders.

Tip: ordering the bars helps the reader. In ggplot, reorder with forcats.

Try it

Solution

library(tidyverse)
library(scales)

usage <- read_csv("data_clean/transport_usage_clean.csv", show_col_types = FALSE)

by_city <- usage %>%
  group_by(city) %>%
  summarise(total_riders = sum(total_monthly_riders), .groups = "drop") %>%
  mutate(city = forcats::fct_reorder(city, total_riders))

ggplot(by_city, aes(x = city, y = total_riders)) +
  geom_col() +
  coord_flip() +
  scale_y_continuous(labels = comma) +
  labs(
    title = "Total riders by city",
    x = NULL,
    y = "Total riders"
  ) +
  theme_minimal(base_size = 12)
If you want "per year" bars, add group_by(city, year) and facet by year.
