Box plot of average delay by transport mode

Goal: compare avg_delay_minutes across transport modes. A box plot shows the median, spread, and outliers.

What a box plot tells you

  • Middle line: median
  • Box: middle 50 percent of values
  • Whiskers: typical ranges
  • Points: outliers
Questions worth considering: which modes are most variable, which are most consistent, and which users risk being late to work most often?

Try it

library(tidyverse)

usage <- read_csv("data_clean/transport_usage_clean.csv", show_col_types = FALSE)

# Make sure transport_mode is treated as a category
# Then build a box plot of avg_delay_minutes by transport_mode

Solution

library(tidyverse)

usage <- read_csv("data_clean/transport_usage_clean.csv", show_col_types = FALSE) %>%
  mutate(transport_mode = as.factor(transport_mode))

ggplot(usage, aes(x = transport_mode, y = avg_delay_minutes)) +
  geom_boxplot(outlier.alpha = 0.4) +
  labs(
    title = "Average delay by transport mode",
    x = NULL,
    y = "Average delay (minutes)"
  ) +
  theme_minimal(base_size = 12)
Optional: reorder modes by median delay using forcats::fct_reorder().

Checks and questions

  • Which mode has the highest median delay?
  • Which mode has the widest spread?
